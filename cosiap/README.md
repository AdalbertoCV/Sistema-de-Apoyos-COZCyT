# Sistema de Apoyos COZCyT (COSIAP)

Este proyecto es realizado por:

* Rafael Uribe Caldera
* Adalberto Cerrillo Vázquez 
* Elliot Axel Noriega
* Narda Viktoria Gómez Aguilera

El sistema COSIAP permite a los solicitantes de apoyos del Consejo Zacatecano de Ciencia, Tecnología e Innovación, realizar sus 
solicitudes de apoyos económicos de manera más eficiente y automatizada, ahorrando tiempos de traslado presencial y documentación física.

Además, le permite a los administradores tener una gestión más efectiva de las solicitudes, dando la oportunidad de crear modalidades y formularios dinámicos personalizados, ver y calificar la documentación, aprobar o rechazar solicitudes, gestionar usuarios entre otras bondades.

# Ejecución del proyecto

Ejecutar desde la raíz para levantar ambos frontend y backend, ejecutar desde el directorio de la API o el frontend para levantarlos individualmente:

* docker compose build
* docker compose up
